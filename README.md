# 🎶 ギターコードメーカー(GuitarChordMaker)

**誰でも簡単にギターのコードを作ることができます！**
**弾き語りや一般公開などにも使ってください**
**使用してみての感想やアドバイスなども心からお待ちしています！**

---

## 🌟 特徴
- **直感的な操作**: 難しい操作はゼロ！歌詞とコードを入力するだけ
- **セーブ機能**: アプリを閉じてもセーブすれば続きからできます
- **PDF/印刷**: アプリ内のボタンからすぐに印刷やPDFでダウンロードできます

## 使用方法
**こちらから開くか、デプロイメントから開いてください**
**htmlなどを自由にアレンジや改造しても構いません**
**https://amekunstudio.github.io/GuitarChordMaker**
